## About This Repository

This repo documents my hands-on practice with Python for data science and machine learning. It started as a learning space and has grown into a portfolio of projects covering:
- Data cleaning and exploratory data analysis (EDA)
- Data visualization
- Supervised & unsupervised machine learning
- Web scraping with Selenium and BeautifulSoup
- Time series analysis and forecasting
- Customer behavior modeling (e.g., churn prediction)

All notebooks are self-contained with explanations, code, and visualizations.

## Core Library Tutorials

These folders contain structured tutorials and examples for foundational libraries.

- **[Matplotlib](Matplotlib/)**  
  Basic to advanced plotting: line plots, scatter plots, subplots, customization, 3D plots, animations.

- **[NumPy](Numpy/)**  
  Array operations, broadcasting, linear algebra, random number generation, performance tips.

- **[Pandas](Pandas/)**  
  Series & DataFrames, data cleaning, merging, grouping, time series handling, I/O operations.

- **[Seaborn](Seaborn/)**  
  Statistical visualizations: heatmaps, pair plots, distribution plots, categorical plots, themes.

## Data Analysis & Machine Learning Projects

| Notebook | Description | Key Techniques |
|----------|-------------|----------------|
| **Airbnb.ipynb** | Exploratory analysis of Airbnb listings (pricing, availability, reviews) | EDA, visualization, geospatial analysis |
| **Flipkart Laptops.ipynb** | Analysis of laptop specifications and pricing from Flipkart | Data cleaning, feature extraction, visualization |
| **car dekho.ipynb** | Used car data analysis from CarDekho platform | Regression, feature engineering, price prediction |
| **customer churn.ipynb** | Customer churn prediction modeling | Classification (Logistic Regression, Random Forest), ROC-AUC |
| **flipkart_reviews_analysis.ipynb** | Sentiment/reviews analysis of Flipkart products | Text preprocessing, sentiment analysis |
| **KNN.ipynb** | K-Nearest Neighbors algorithm from scratch and using Scikit-learn | Classification, distance metrics |
| **k-means clustering.ipynb** | K-Means clustering implementation and applications | Unsupervised learning, elbow method, silhouette score |
